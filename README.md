# Kildekritik

Kildekritik er et browserbaseret analyseværktøj til gymnasiets historiefag. Appen er udviklet til tekstnær og kildekritisk analyse, hvor elever kan markere kildetekst i kategorier, skrive kommentarer og samle markeringer i en tidslinje.

## Brug

Installer afhængigheder og start udviklingsserveren:

```bash
npm install
npm run dev
```

Produktionsbuild:

```bash
npm run build
```
